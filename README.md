## Code Block with Prism JS Highlight

Gutenberg block for code Highlighting.

## Features

- Choose a theme in settings of plugin.
- show or hide the numbers of lines
- Highlight lines
- Choose a language :
  - markup
  - css
  - clike
  - javascript
  - actionscript
  - aspnet
  - bash
  - c
  - csharp
  - diffdocker
  - go
  - graphql
  - http
  - java
  - json
  - json5
  - jsonp
  - kotlin
  - makefile
  - markdown
  - markup-templating
  - mongodb
  - nginx
  - ocaml
  - php
  - python
  - jsx
  - tsx
  - ruby
  - rust
  - scss
  - smarty
  - sql
  - swift
  - toml
  - twig
  - typescript
  - vim
  - warpscript
  - wasm
  - yaml
